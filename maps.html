Maps

🌐Languages: Java, html/CSS, Javascript
📅Date: Spring, 2018
🔗Link: Code available upon request

For a class project, I created a web application that maps the entire state of Rhode Island.

Maps is written in scratch from Java, and implements a KD Tree to search for the closest location to a clicked point on the map, a Trie structure to implement autocorrect on street names, and uses Dijkstra's algorithm with A* search to find the shortest path between points. Users can input latitude and longitude coordinates or two intersecting streets, and they can pan and zoom the map. There is caching on the frontend for reducing drawing and on the backend for minimizing database queries. 🗺️
