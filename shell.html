UNIX Shell

🌐Language: C
📅Date: Fall, 2017
🔗Link: Code available upon request

One of my most exciting projects in my Systems class was creating a UNIX shell in C from scratch. This terminal had many key features, including:

  • Processing commands such as cd, rm, exit
  • Handling different signals, including stop and kill
  • Executing processes and dealing with foreground and background job control
  • Checking for error conditions at all stages of command line parsing

Working on this project deepened my knowledge of how computers work. 🐚
